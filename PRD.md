## 1. 개요

### 1.1 배경
시중의 AI 녹음 요약 서비스(클로바 노트, Otter.ai, Fireflies 등)는 월 $10~30 수준으로 비싸고, 커스터마이즈가 어렵다. 직접 만들어 팀 내부에서 원가 수준으로 운영한다.

### 1.2 목적
갤럭시 폰에서 녹음한 파일을 업로드하면 STT 변환 → 화자분리 → AI 회의록/요약 생성 → 저장 및 이메일 발송까지 자동으로 처리해주는 팀용 웹앱을 만든다.

### 1.3 목표 사용자
- 본인 (어드민)
- 팀원 5~10명 (초대된 사용자)

## 2. 핵심 기능 (MVP Scope)

### 2.1 인증 및 팀 관리
| 기능 | 상세 |
|------|------|
| Google 로그인 | OAuth 2.0, 별도 비밀번호 없음 |
| 접근 제어 | 어드민이 허용한 이메일만 입장 가능 |
| 팀원 초대 | 이메일 초대 → 링크 클릭 → 자동 승인 |
| 어드민 권한 | 팀원 초대/삭제, 전체 이력 조회 |

### 2.2 파일 업로드
| 기능 | 상세 |
|------|------|
| 지원 형식 | m4a (갤럭시 기본), mp3, wav |
| 최대 크기 | 2GB |
| 최대 길이 | 4시간 |
| 업로드 방식 | 브라우저 파일 선택 or 드래그앤드롭 |
| 진행률 표시 | 업로드 % 실시간 표시 |
| 메타데이터 입력 | 업로드 시 아래 항목 입력 |

**업로드 시 입력 필드**
```
제목 *         (직접 입력, 예: "6월 팀 주간회의")
유형           팀 회의 / 고객 미팅 / 1:1 면담 / 기타
공개 범위 *    팀 공유 / 나만 보기
출력 형식 *    회의록 / 요약
날짜           자동 (업로드 시각 기준, 수정 가능)
```

### 2.3 STT + 화자분리
| 항목 | 내용 |
|------|------|
| STT API | 리턴제로 (RTZR) |
| 모델 | Sommers (한국어 특화) |
| 화자분리 | 최대 10인, 자동 감지 |
| 타임스탬프 | 발화별 시작 시각 기록 |
| 출력 | 화자 ID + 시작시각 + 발화 텍스트 |

**STT 결과 예시**
```
[00:02:14] 화자1: 이번 분기 매출 목표는 어떻게 잡을까요?
[00:02:28] 화자2: 저는 전 분기 대비 20% 성장을 목표로 하면...
[00:03:05] 화자1: 현실적으로 가능한 수치인지 검토가 필요할 것 같고요.
```

### 2.3.1 화자 이름 지정
STT 완료 후 GPT 정리 전에 화자 이름을 매핑하는 단계를 거친다.

| 항목 | 내용 |
|------|------|
| 표시 시점 | STT 완료 직후, AI 정리 전 |
| 입력 방식 | 화자별 이름 텍스트 입력 |
| 기본값 | 화자1, 화자2, 화자3 ... (미입력 시 그대로 사용) |
| 반영 범위 | STT 전문, 회의록/요약, 이메일, Notion 페이지 전체 |

**화자 이름 지정 UI**
```
┌─────────────────────────────┐
│  화자 이름을 지정해주세요    │
│  (건너뛰면 화자1/2/3 유지)  │
│                             │
│  화자1 → [홍길동      ]     │
│  화자2 → [김철수      ]     │
│  화자3 → [이영희      ]     │
│                             │
│  [건너뛰기]  [적용 → 생성]  │
└─────────────────────────────┘
```

**이름 적용 후 출력 예시**
```
[00:02:14] 홍길동: 이번 분기 매출 목표는 어떻게 잡을까요?
[00:02:28] 김철수: 저는 전 분기 대비 20% 성장을 목표로 하면...
[00:03:05] 홍길동: 현실적으로 가능한 수치인지 검토가 필요할 것 같고요.
```

### 2.4 AI 정리
| 항목 | 내용 |
|------|------|
| LLM | GPT-5.5 (OpenAI API) |
| 입력 | STT 전문 텍스트 (화자 이름 반영) |
| 출력 형식 | 업로드 시 선택한 형식 |

**회의록 출력 형식**
```
📋 회의록

일시: 2026-06-06
참석자: 홍길동, 김철수, 이영희
유형: 팀 회의

📌 안건
- ...

💬 주요 내용
[홍길동] ...
[김철수] ...

✅ 결정사항
- ...

📌 액션아이템
- [ ] 홍길동: ... (기한: ?)
- [ ] 김철수: ...
```

**요약 출력 형식**
```
📝 요약

핵심 요약 (3줄)
- ...
- ...
- ...

주요 논의사항
...

결론
...
```

### 2.5 결과물 저장
| 대상 | 저장 내용 |
|------|---------|
| Cloudflare R2 | 원본 오디오 파일 (m4a) — 타임스탬프 재생용 |
| Supabase DB | 메타데이터, STT 결과, AI 결과물, 처리 이력 |
| Notion | 회의록/요약 페이지 자동 생성 |

> Google Drive 연동 제외 — Cloudflare R2로 오디오 저장 대체, 저장소 단일화

**Notion 폴더 구조**
```
📁 AI 회의록
├── 📁 팀 공유
│   └── [날짜] 제목 (업로더)
└── 📁 개인
    └── [사용자명]
        └── [날짜] 제목
```

### 2.6 타임스탬프 기반 오디오 재생
| 기능 | 상세 |
|------|------|
| STT 결과 화면 | 타임스탬프 클릭 시 해당 구간부터 재생 |
| 오디오 소스 | Cloudflare R2에서 Range request 스트리밍 |
| 플레이어 | 인라인 미니 플레이어 (재생/일시정지/구간 표시) |

### 2.7 이메일 발송
| 항목 | 내용 |
|------|------|
| 발송 시점 | 처리 완료 즉시 |
| 수신자 | 업로드한 본인 |
| 내용 | 회의록/요약 전문 + Notion 링크 |
| 발송 수단 | 네이버 웍스 SMTP (회사 계정) |

**이메일 발송 설정**
```
SMTP 서버: smtp.worksmobile.com
포트:      465 (SSL)
계정:      회사 네이버 웍스 이메일
비밀번호:  앱 비밀번호 (웍스 관리자 콘솔 발급)
라이브러리: Nodemailer
```

**이메일 구조**
```
발신자: 회의록봇@회사.com
제목:   ✅ [6월 팀 주간회의] 회의록이 완성되었습니다

본문:
회의록 전문 내용
...

---
📎 Notion에서 보기: [링크]
📁 원본 파일: [Cloudflare R2 링크]
```

### 2.8 처리 이력
| 기능 | 상세 |
|------|------|
| 팀 공유 목록 | 팀 공유로 올린 전체 목록 |
| 내 녹음 목록 | 나만 보기로 올린 것 |
| 표시 정보 | 제목, 유형, 업로더, 길이, 날짜, 처리 상태 |
| 결과 재조회 | 과거 처리 결과 언제든 다시 볼 수 있음 |

## 3. 처리 흐름

```
[사용자]
  │
  ├── 1. Google 로그인
  ├── 2. 파일 선택 + 메타데이터 입력
  ├── 3. 업로드 (브라우저 → Cloudflare R2 직접)
  │         ↓ 업로드 완료
  ├── 4. 처리 시작 (Vercel API Route)
  │     ├── 리턴제로 STT 요청 (비동기)
  │     └── Job ID 반환 → Supabase에 저장
  │
  ├── 5. STT 완료 → 화자 이름 지정 UI 표시
  │     └── 이름 확정 → GPT-5.5 회의록/요약 생성
  │               ↓
  │         Notion 페이지 생성
  │         이메일 발송 (네이버 웍스 SMTP)
  │
  └── 6. 완료
        ├── 결과 화면 (STT 전문 + 회의록/요약)
        ├── 타임스탬프 클릭 → 오디오 재생
        └── 이메일 수신 확인
```

## 4. 화면 목록

| 화면 | 경로 | 설명 |
|------|------|------|
| 로그인 | `/login` | Google 로그인 버튼 |
| 홈 / 업로드 | `/` | 파일 업로드 + 메타데이터 입력 |
| 처리 중 | `/processing/[id]` | 단계별 진행 상태 표시 |
| 화자 이름 지정 | `/processing/[id]/speakers` | STT 완료 후 이름 매핑 |
| 결과 보기 | `/result/[id]` | STT 전문 + 회의록/요약 + 오디오 플레이어 |
| 이력 목록 | `/history` | 팀 공유 / 내 녹음 탭 분리 |
| 어드민 | `/admin` | 팀원 초대 / 관리 |
